Featured app from Init4Labs

HowManySleeps

HowManySleeps is a friendly countdown app for upcoming dates, trips, routines and important moments. It is designed to keep event planning simple while still supporting repeat rules, widgets, reminders and shared timetable links.

Get support Read privacy policy

Countdowns that feel simple

Create one-off or repeating events and see them as clear “sleeps until” countdowns.

Built for real use

Widgets, reminders, palettes, optional icons and PIN-protected settings are all built around everyday family and learning needs.

Share-friendly

Widgit Online links can be shared into the app so new events are prefilled more quickly.

What the app supports

  • One-off and repeating countdown events
  • Color palettes and custom colors
  • Optional event icons and Widgit links
  • Home Screen and Lock Screen widgets
  • Daily reminder notifications
  • Superuser PIN and standard user permission modes
